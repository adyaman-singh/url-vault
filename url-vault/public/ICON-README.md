# Icon Instructions

The manifest.json expects icon files in the public folder. You can:

## Option 1: Create Simple Icons

Use any image editor to create PNG files with these dimensions:

- icon-16.png (16x16 pixels)
- icon-48.png (48x48 pixels)
- icon-128.png (128x128 pixels)

## Option 2: Use Emoji as Icon (Quick)

For testing, you can use an online emoji-to-icon converter:

1. Go to https://favicon.io/emoji-favicons/
2. Search for "link" or "lock" emoji
3. Download and rename the icons

## Icon Design Tips

- Keep it simple and recognizable at small sizes
- Use high contrast for visibility
- Match the light, minimal aesthetic of the extension
- Consider using the app's primary blue color
